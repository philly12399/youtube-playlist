# 更新日誌

此專案的所有重大更新都將記錄於此文件中。

## [1.0.0] - 2026-05-13

### 新增功能
- **雙重輸入模式**：支援「文字」（快速解析）與 「JSON」（進階結構）兩種播放清單輸入方式。
- **智慧解析**：自動從各種文字格式中提取 YouTube 影片 ID 與時間軸資料。
- **播放清單庫**：可將多組播放清單儲存至瀏覽器的本地儲存 (LocalStorage)，並支援影片縮圖預覽。
- **純音訊模式**：一鍵切換遮罩以隱藏影片內容，適合當作背景音樂聆聽。
- **快捷鍵支援**：支援播放/暫停 (Space)、上一首 (Z)、下一首 (X) 及音訊模式切換 (C)。
- **匯入/匯出系統**： 
    - 可將個別播放清單匯出為 JSON 檔案。
    - 可將整個歌單庫打包為 ZIP 壓縮檔（使用 JSZip 實作）。
    - 支援從 JSON 檔案匯入至歌單庫或目前播放清單。
- **可摺疊介面**：播放清單與歌單庫側邊欄皆可收合，以最大化影片觀看區域。
- **響應式設計**：使用 Pure.css 網格系統，完美適配行動裝置佈局。
- **使用者體驗優化**： 
    - 提供 Toast 訊息通知功能。
    - 標題字體大小會根據容器寬度自動調整。
    - 瀏覽器重新整理後仍能保持最後的播放狀態。
- **GitHub 整合**：頁尾包含專案連結與按鈕，提高開源專案可見度。
